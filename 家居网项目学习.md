#### 暴露模块

- [webpack-chain](https://github.com/neutrinojs/webpack-chain) vue中使用chainWebpack配置项

- configureWebpack配置项 合并多个配置文件

```javascript
configureWebpack: (config) => {
  if (!isProduction) {
    let pluginsDev = [
      new vConsolePlugin({
        filter: [],
        enable: true
      })
    ]
    config.plugins = [...config.plugins, ...pluginsDev]
  }
},
```
